# Code of Conduct

## The Gist
Be cool. Don't be a jerk. We're here to learn and build a solid resource.

## Zero Tolerance
We have zero tolerance for:
*   Harassment of any kind.
*   Racism, sexism, homophobia, transphobia, or ableism.
*   Spamming or trolling.

## Enforcement
If you act up, you get banned. Simple as that.

## Scope
This code of conduct applies to all spaces managed by this project, including GitHub issues, PRs, and discussions.
