# Security Policy

## ⚠️ Disclaimer

This repository is a collection of tools for **educational and authorized security testing purposes only**. 

By using this repository, you agree that:
1.  You will not use these tools for illegal purposes.
2.  You will not target systems you do not own or have explicit permission to test.
3.  The maintainers are not responsible for your actions.

## Reporting a Vulnerability

If you find a security issue **within this repository** (e.g., a malicious link that slipped through our checks), please do NOT open a public issue.

Instead, email us privately or DM the maintainer.

## Supply Chain Security

We do our best to verify that linked repositories are the original authors' and not malicious forks. However, always **audit the code** before running it on your machine.

**Trust, but verify.**
